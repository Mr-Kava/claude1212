# Aetheris — симуляция мира с эволюцией существ

Проект состоит из двух взаимодополняющих частей:

1. **GUI-приложение** (`main.py` + `app/`) — настольное окно на pygame с картой 50×50,
   существами в реальном времени и боковой панелью статистики.
2. **CLI-симулятор** (`simulation.py`) — самодостаточный консольный скрипт
   без зависимостей для воспроизводимых прогонов.

Обе части используют общий мир: 7 типов ландшафта, 8 видов существ,
5-параметровую эволюцию, природные катастрофы.

---

## 🎮 Быстрый старт: GUI-приложение

### Установка

```bash
python3 -m pip install -r requirements.txt
```

Требуется только **pygame ≥ 2.5** и Python ≥ 3.10.

### Запуск

```bash
python3 main.py
```

Откроется окно **940×600** с картой слева и панелью статистики справа.

### Управление

| Клавиша | Действие |
|---|---|
| **Space** | пауза / продолжить |
| **↑ / ↓** | увеличить / уменьшить скорость симуляции (от ×0.25 до ×8) |
| **R** | рестарт со случайным новым seed |
| **S** | сохранить отчёт и статистику в `./output_app/` |
| **Esc / Q** | выйти (отчёт сохраняется автоматически) |

### Запуск с параметрами

```bash
python3 main.py --seed 7 --tps 10            # фиксированный seed, 10 тиков/сек
python3 main.py --headless --ticks 300       # без GUI, только прогон+отчёт
python3 main.py --headless --ticks 500 --seed 42 --out ./my_run
```

### Что отображается в окне

- **Слева** — карта мира, каждый тайл закрашен в цвет своего ландшафта,
  яркость тайла пропорциональна количеству пищи в нём.
- **Существа** — цветные точки. Цвет соответствует виду.
- **Справа** — панель с:
  - текущим тиком и seed'ом,
  - популяцией каждого вида + цветовая легенда,
  - максимальным и средним поколением,
  - средними генами доминирующего вида (показывает эволюцию в реальном времени),
  - последними катастрофами (засуха, мороз, эпидемия, метеорит).

## 🧬 Правила эволюции

При каждом размножении (нужна энергия и партнёр рядом):
- гены родителей **усредняются**,
- добавляется лёгкий **гауссов дрейф** (σ=0.15) к каждому гену,
- с шансом **4%** к гену прибавляется **большая мутация** (σ=1.2),
- значение обрезается в `[0.1, 12]`.

Естественный отбор работает через выживание: неудачные гены элиминируются
голодом, хищниками и катастрофами.

Подробнее — [`docs/evolution.md`](docs/evolution.md).

## 🌪️ Природные катастрофы

С вероятностью **1.2%** на каждый тик случается одно из:
- **Засуха** — биомасса в пустынях и равнинах падает до 20%
- **Великий мороз** — массовая гибель в тундре/горах для не-адаптированных
- **Эпидемия** — каждое 5-е существо ослабевает
- **Метеорит** — в случайной точке радиуса 5 всё уничтожено

## 📊 Отчёт о симуляции

При выходе из GUI или нажатии **S** в `./output_app/` сохраняется:

- `report.md` — какие виды выжили / вымерли, почему, как изменилась экосистема,
  какие гены сдвинулись.
- `stats.json` — полная история популяций и средних генов по тикам.

Пример референсного прогона CLI-версии — в [`docs/report.md`](docs/report.md).

```
Creature ──genes──► speed, vision, aggression, intellect, stamina
                 ├── species_name → SPECIES[name]
                 └── energy, age, generation
```

## ⚠️ Известные ограничения

- GUI протестирован на машине пользователя — в Tembo-песочнице (Docker без X-сервера)
  pygame запустить невозможно, но синтаксис всего кода проверен `python3 -m compileall`,
  а вся симуляционная логика прогнана через `--headless` режим.
- Для очень больших популяций (>2000 существ) производительность может проседать —
  в этом случае нажми **↓** в окне, чтобы снизить TPS.

## CLI-симулятор: команды

```bash
python3 simulation.py                            # дефолт: 300 тиков, seed=42
python3 simulation.py --ticks 500 --seed 7      # больше тиков, другой seed
python3 simulation.py --out /path/to/output     # папка для результатов
python3 simulation.py --quiet                   # без прогресса в консоли
```

Выходные файлы (в `--out`):
| Файл | Что |
|---|---|
| `world_map_initial.txt` | ASCII-карта на старте |
| `world_map_final.txt` | ASCII-карта в финале |
| `food_heatmap.txt` | Heatmap биомассы |
| `population_chart.txt` | ASCII-графики популяций по видам |
| `stats.json` | Полная история |
| `report.md` | Автоматический отчёт |
