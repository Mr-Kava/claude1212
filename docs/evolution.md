# Правила эволюции

Эволюция в Aetheris работает как **дарвиновский отбор по поколениям**:
гены передаются от родителей, случайно мутируют, а среда выбраковывает
неудачные сочетания.

## 1. Гены

У каждого существа 5 эволюционирующих параметров (`EVOLVABLE`):

| Ген | Диапазон | Влияние |
|---|---|---|
| `speed` | 0.1 – 12 | Скорость движения (клеток за тик) и шанс убийства/побега |
| `vision` | 0.1 – 12 | Радиус восприятия мира (поиск пищи, жертв, угроз) |
| `aggression` | 0.1 – 12 | Готовность нападать; шанс инициировать охоту |
| `intellect` | 0.1 – 12 | Качество движения (биас к лучшей клетке vs. рандом) и охоты |
| `stamina` | 0.1 – 12 | Снижение метаболизма; шанс выжить при катастрофах |

Базовый максимум 12 (вид может эволюционировать выше своего стартового
значения, но не выше 12).

## 2. Размножение

Существо может размножаться когда:

```
energy ≥ repro_threshold (определяется видом, обычно 45–110)
age ≥ 5
рядом (в радиусе 2) есть партнёр того же вида с энергией ≥ 0.7·threshold
```

При успехе:

1. Из генов обоих родителей берётся **среднее**.
2. К каждому гену добавляется **гауссов дрейф** σ=0.15.
3. С шансом **4%** на каждый ген — **мутация** σ=1.2 (заметный сдвиг).
4. Ген обрезается в `[0.1, 12]`.
5. Энергия родителей падает: автора на 55%, партнёра на 70%.

```python
# Из simulation.py:
avg = (parent_a.gene + parent_b.gene) / 2
avg += gauss(0, 0.15)                 # лёгкий дрейф
if random() < 0.04:                   # 4% шанс мутации
    avg += gauss(0, 1.2)              # большой сдвиг
avg = clamp(avg, 0.1, 12)
```

## 3. Естественный отбор

Отбор не явный — он **эмерджентный** через выживание:

- Существа с низкой `stamina` тратят больше энергии и быстрее умирают от голода.
- Существа с низкими `speed/vision/intellect` хуже находят пищу и легче ловятся.
- Хищники с низкой `aggression` или `intellect` чаще промахиваются.
- Гены, дающие плохое выживание в текущем биоме, элиминируются за несколько поколений.

## 4. Адаптация к климату

Каждый ландшафт даёт **метаболический штраф** (см. `docs/world.md`). Виды,
у которых высокая `stamina`, переносят жёсткие климаты (пустыни, тундра, горы)
без катастрофических потерь энергии. Это создаёт давление отбора в пользу
выносливости в этих биомах.

Кроме того, у каждого вида есть жёсткий **список запрещённых ландшафтов**
(`forbidden`) — он не эволюционирует. Aquor никогда не выйдет из воды,
Grazor никогда не освоит горы.

## 5. Конкуренция за ресурсы

Все травоядные/всеядные едят из одной общей биомассы клетки. Когда популяция
травоядных превышает скорость регенерации биомассы (`regrowth`), они
**подавляют сами себя** через голод.

Хищники конкурируют за общий пул жертв; если их слишком много, они
быстро истощают добычу и **коллапсируют** — это классическая модель
Лотки—Вольтерры.

## 6. Шанс мутации — детали

| Параметр | Значение |
|---|---|
| `GLOBAL_MUTATION_RATE` | **0.04** (4% на каждый ген) |
| Стандартный дрейф каждого гена | σ = 0.15 (всегда) |
| Размер мутации | σ = 1.2 (только при срабатывании 4%) |

Это даёт **в среднем 1 заметную мутацию на 5 рождений** (5 генов × 4%).
За 200–300 тиков и сотни поколений виды эволюционируют ощутимо
(пример из реального прогона: у Cryon `intellect` выросло 5 → 5.31,
у Shakrit `vision` выросло 6 → 6.57; у Terrax `speed` выросло 2 → 2.79).

## 7. Изменение по поколениям

После каждого тика симуляция считает **среднее значение каждого гена по
виду** и пишет в `gene_history[species]`. Это позволяет потом построить
эволюционный график — например, увидеть, как **vision хищника растёт**
вместе с тем, как у их жертвы растёт **speed** (гонка вооружений).

## 8. Эволюционные сценарии (что мы наблюдаем)

- **r-стратеги** (Mycor, Grazor): много потомства, низкая защита — выживают только
  если хищники истреблены или удалены.
- **K-стратеги** (Terrax, Cryon): мало потомства, высокая стамина — выживают
  даже в катастрофах за счёт индивидуальной живучести.
- **Гипер-специалисты** (Aquor — только вода, Skywing — нужны раненые) уязвимы
  к узким катастрофам (засуха, конец популяционных вспышек).
- **Generalist omnivores** (Cryon, Shakrit) показывают самую стабильную
  динамику.
